# 一、	簡介
　　可以利用此APP比較蝦皮跟淘寶的商品，減少需要開啟多個APP或頁面，來找尋同個商品之不同價格及資訊。

# 二、User Story
身為一個學生	我希望可以查到便宜又實惠的商品價格，這樣才能省錢。
身為一個上班族	我希望能夠多方比較，這樣可以避免吃虧。
身為一個管理者	我希望能給大眾更多元的比價APP，這樣能使大眾找尋商品更加便利。

# 三、利害關係人目標表
顧客	
1.	可以買到CP值高的好商品
2.	多方比較可避免買到瑕疵品。
3.	貨比三家不吃虧。
4.	避免遇到黑心商人或是詐騙集團。

系統管理者	
1.	提供好的APP給大眾。
2.	提供更多商品資訊服務。

商品店家	
1.	可以利用此APP，查詢其他店家的商品資訊。
2.	比較其他店家的銷售方式。
